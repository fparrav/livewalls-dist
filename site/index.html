LiveWalls

Animated wallpapers for macOS — multi‑monitor, SwiftUI, open source.

Download Latest View on GitHub

Quick Install (Homebrew)

brew tap fparrav/livewalls
brew install --cask livewalls

Note: first launch may require allowing the app in System Settings → Privacy & Security.

Features

  • Multi‑monitor wallpapers (one window per screen)
  • Video playlist with transitions
  • Per‑monitor management
  • Launch at login and auto‑resume
  • In‑app updates via Sparkle

Updates

LiveWalls checks for updates using Sparkle.
