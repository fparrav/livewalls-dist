Pick the installation that fits your workflow

Whether you prefer a drag-and-drop DMG or a fully scripted setup, LiveWalls is easy to install and update.

Download the DMG

  1. Grab the latest release from GitHub.
  2. Drag LiveWalls.app to Applications.
  3. Launch it and start curating your wallpapers.
First launch on macOS

Because LiveWalls is distributed unsigned, remove the quarantine flag once:

sudo xattr -rd com.apple.quarantine /Applications/LiveWalls.app

Alternatively, use System Settings ‚Üí Privacy & Security ‚Üí Open Anyway.

Install with Homebrew

Best for power users who want painless upgrades.

brew tap fparrav/livewalls
brew install livewalls
  • No manual quarantine dance.
  • Update with brew upgrade.
  • Tap maintained alongside every release.

Always up to date with Sparkle

LiveWalls ships with Sparkle 2 so you can receive delta updates directly inside the app. Prefer manual checks? Subscribe to the appcast at any time.
